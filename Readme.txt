This is an online marketplace demonstration. Steps to run this on localhost.
1. You need to install a software that provides a server for apache and mysql. 
2. XAMMP, WAMMP are some of the softwares that provide this.
3. Run XAMPP and start the Apache and MYSQL servers in it.
4. Extract this zip folder into xammp/htdocs folder
5. Load the marketplaceschema given in localhost:portnumber/phpmyadmin
5. Go to the web browser and type localhost:portnumber/MarketPlace
6. The app will start and will display index.php
